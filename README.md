# Docker: Hello Captain! 🐳

This is a simple Docker project that prints “Hello, Captain!” when the container is run.

## 🛠 Requirements

- Docker installed

## 🚀 How to Run

```bash
# Build the Docker image
docker build -t hello-captain .

# Run the container
docker run hello-captain
